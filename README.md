# Digital Nurture Java FSE

This repository contains the hands-on exercises completed as part of the Cognizant Digital Nurture Program – Java Full Stack Engineer (Java FSE).

## Week 1

### Design Patterns and Principles

#### Exercise 1: Singleton Pattern
- Implemented Singleton Design Pattern.
- Ensures only one instance of the Logger class exists.

#### Exercise 2: Factory Method Pattern
- Implemented Factory Method Design Pattern.
- Demonstrates object creation using factory classes.

### Data Structures and Algorithms

#### Exercise 1: E-commerce Search Platform
- Implemented Linear Search.
- Implemented Binary Search.
- Compared search performance and efficiency.

#### Exercise 2: Financial Forecasting
- Implemented recursive forecasting algorithm.
- Demonstrated optimization concepts using recursion.

### PL/SQL Programming

#### Exercise 1: Control Structures
- Implemented conditional statements.
- Used loops and control flow statements.

#### Exercise 2: Stored Procedures
- Created and executed stored procedures.
- Demonstrated parameterized procedure calls.

### Technologies Used

- Java
- Object-Oriented Programming
- Design Patterns
- Data Structures & Algorithms
- SQL / PL-SQL
- Git & GitHub
