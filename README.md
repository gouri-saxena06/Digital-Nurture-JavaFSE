# Digital Nurture Java FSE

This repository contains the hands-on exercises completed as part of the Cognizant Digital Nurture Program – Java Full Stack Engineer (Java FSE).

---

# Week 1

## Design Patterns and Principles

### Exercise 1: Singleton Pattern
- Implemented Singleton Design Pattern.
- Ensures only one instance of the Logger class exists.

### Exercise 2: Factory Method Pattern
- Implemented Factory Method Design Pattern.
- Demonstrates object creation using factory classes.

## Data Structures and Algorithms

### Exercise 1: E-commerce Search Platform
- Implemented Linear Search.
- Implemented Binary Search.
- Compared search performance and efficiency.

### Exercise 2: Financial Forecasting
- Implemented recursive forecasting algorithm.
- Demonstrated optimization concepts using recursion.

## PL/SQL Programming

### Exercise 1: Control Structures
- Implemented conditional statements.
- Used loops and control flow statements.

### Exercise 2: Stored Procedures
- Created and executed stored procedures.
- Demonstrated parameterized procedure calls.

# Week 2

## JUnit 5

### Exercise 1: Setting Up JUnit
- Configured JUnit 5 in a Maven project.
- Created and executed basic unit tests.

### Exercise 3: Assertions in JUnit
- Implemented assertion methods such as:
  - assertEquals()
  - assertTrue()
  - assertFalse()
  - assertNull()
  - assertNotNull()

### Exercise 4: Arrange-Act-Assert (AAA) Pattern
- Created test cases using the AAA testing approach.
- Implemented setup and teardown methods using:
  - @BeforeEach
  - @AfterEach

## Mockito

### Exercise 1: Mocking and Stubbing
- Created mock objects using Mockito.
- Implemented stubbing with when().thenReturn().

### Exercise 2: Verifying Interactions
- Verified method invocations using Mockito verify().
- Tested service interactions independently.

## SLF4J Logging

### Exercise 1: Logging Error Messages and Warning Levels
- Implemented logging using SLF4J.
- Generated INFO, WARN, and ERROR log messages.

## Spring Core & Maven

### Exercise 1: Configuring a Basic Spring Application
- Created a basic Spring Core application.
- Configured beans using XML configuration.

### Exercise 2: Implementing Dependency Injection
- Implemented Dependency Injection using Spring.
- Configured bean dependencies.

### Exercise 4: Creating and Configuring a Maven Project
- Created a Maven-based Java project.
- Managed dependencies using pom.xml.

## Additional Recommended Exercises

### Exercise 5: Configuring Spring IoC Container
- Configured and managed Spring beans using the IoC container.

### Exercise 7: Constructor and Setter Injection
- Implemented Constructor Injection.
- Implemented Setter Injection.

### Exercise 9: Creating a Spring Boot Application
- Created a basic Spring Boot application.
- Executed the application using Spring Boot Starter.
